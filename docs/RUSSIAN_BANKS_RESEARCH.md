# Исследование: Российские банки с ежемесячными кешбэками (2026)

**Дата исследования:** 2026-02-07
**Цель:** Определить топ-банки для MVP приложения BuyWhyWhy

---

## Итоговые рекомендации для MVP

### Топ-5 банков для включения в MVP:

1. **Сбербанк (СберСпасибо)** - лидер рынка, наибольшая база пользователей
2. **Т-Банк (Тинькофф)** - популярен среди tech-savvy аудитории
3. **Альфа-Банк** - сильные категории кешбэка
4. **ВТБ** - один из крупнейших банков
5. **Райффайзен Банк** - стабильная программа кешбэка

**НЕ включаем в MVP:**
- Открытие (отсутствуют актуальные данные о кешбэках)
- Росбанк (интеграция с Т-Банком, неясная судьба программы)

## Детальная информация по банкам

### 1. Сбербанк (СберСпасибо) ⭐⭐⭐⭐⭐

**Приоритет для MVP:** КРИТИЧЕСКИЙ (must-have)

**Программа лояльности:** СберСпасибо

**Механика выбора категорий:**
- С 1 февраля 2026 года участники могут активировать категории на месяц
- **Без подписок:** 3 категории
- **СберПрайм:** 5 категорий
- **СберПрайм+:** 5 категорий
- **Премиальные клиенты:** до 6 категорий

**Процент кешбэка:**
- Стандартный: 0.5% на все покупки
- Повышенный: до 10% в выбранных категориях (зависит от статуса)

**Частота обновления категорий:** Ежемесячно

**Источники:**
- [Сбер предложил выбрать категории кешбэка на февраль](https://bankinform.ru/news/140503)
- [Категории на январь 2026](https://bankinform.ru/news/140305)

### 2. Т-Банк (Тинькофф) ⭐⭐⭐⭐⭐

**Приоритет для MVP:** КРИТИЧЕСКИЙ (must-have)

**Программа:** Встроенная программа кешбэка

**Механика выбора категорий:**
- Выбор категорий в конце предыдущего месяца (январь для февраля)
- **Стандартный тариф:** 4 категории с кешбэком до 15%
- **Подписка Pro:** 8 категорий
- **Premium:** 8 категорий + увеличенный лимит
- **Всегда доступно:** 1% на все покупки от 100₽

**Лимиты кешбэка:**
- Стандарт: 3,000₽ за расчетный период
- Pro: 5,000₽
- Premium: 30,000₽

**Примеры категорий:** такси, аптеки, развлечения, одежда, транспорт, рестораны

**Частота обновления:** Ежемесячно

**Источники:**
- [Кэшбэк по карте Т-Банка](https://www.tbank.ru/bank/help/debit-cards/tinkoff-black/earn-with-card/)
- [Повышенный кэшбэк за обычные покупки](https://www.tbank.ru/bank/help/debit-cards/tinkoff-black/earn-with-card/regular-cashback/)
- [Как работает кэшбэк Т-Банк](https://www.banki.ru/news/daytheme/?id=11001800)

### 3. Альфа-Банк ⭐⭐⭐⭐

**Приоритет для MVP:** ВЫСОКИЙ (should-have)

**Программа:** Альфа кешбэк

**Механика выбора категорий:**
- Персонализация программы кешбэка
- **Стандартный пользователь:** до 3 категорий повышенного кешбэка
- **Премиум клиенты:** больше категорий (точное число не указано)

**Процент кешбэка:**
- Базовый: 1.5% на все покупки
- Повышенный: до 5-10% в выбранных категориях

**Частота обновления:** Ежемесячно (февраль 2026 активен)

**Источники:**
- [Категории кэшбэка Альфа-Банк на февраль 2026](https://xvestor.ru/bonuses/alfa/alfa-bank-cashback-category)

### 4. ВТБ ⭐⭐⭐⭐

**Приоритет для MVP:** ВЫСОКИЙ (should-have)

**Программа:** ВТБ Мультикарта / Cashback карты

**Механика выбора категорий:**
- Категории повышенного кешбэка доступны для выбора
- Количество категорий зависит от типа карты

**Примеры категорий (февраль 2026):**
- **Книги и канцтовары:** 15% (лимит 2,000₽)
- Другие категории: продукты, рестораны, АЗС, аптеки

**Процент кешбэка:**
- Базовый: 1% на все покупки
- Повышенный: до 15% в выбранных категориях

**Частота обновления:** Ежемесячно

**Тренд 2026:** ВТБ указал на геймификацию и персонализацию кешбэков

### 5. Райффайзен Банк ⭐⭐⭐

**Приоритет для MVP:** СРЕДНИЙ (nice-to-have)

**Программа:** Кэшбэк-карта Райффайзен

**Механика:**
- **Базовый:** 1% кешбэк на все покупки без лимитов
- **С подпиской "Райффайзен Подписка" (299₽/мес):**
  - Повышенный кешбэк в категориях:
    - Супермаркеты
    - Такси
    - Рестораны и кафе
    - Фастфуд

**Процент кешбэка:**
- Базовый: 1%
- Повышенный (с подпиской): точный % не указан

**Начисление:** 2-5 число следующего месяца

**Исключения:** НЕ начисляется за:
- Оплату мобильной связи
- Коммунальные платежи

**Источники:**
- [Кэшбэк карта Райффайзен 2026](https://bankinform.ru/services/plasticcards/info/504-91-45)
- [Какой кешбэк в Райффайзенбанке](https://fpa.ru/info/kak-nachisljaetsja-kjeshbjek-v-rajffajzenbanke/)

### 6. Банк ДОМ.РФ ⭐⭐

**Приоритет для MVP:** НИЗКИЙ (можно отложить на Phase 2)

**Программа:** Квартальные категории кешбэка

**Механика:**
- Выбор **ежеквартально** (не ежемесячно!)
- С 1 января 2026 в приложении выбирайте **4 категории**
- Действуют весь квартал

**Частота обновления:** Ежеквартально (реже чем у конкурентов)

**Почему низкий приоритет:**
- Квартальное обновление (не месячное) - менее гибко
- Меньшая база пользователей
- Меньше информации о конкретных категориях

**Источники:**
- [Кешбэк Банк ДОМ.РФ](https://domrfbank.ru/cashback/)

### ❌ НЕ включаем в MVP

#### Банк Открытие
**Причина:** Отсутствуют актуальные данные о программе кешбэка на 2026
**Статус:** Возможно включение в Phase 2 после дополнительного исследования

#### Росбанк
**Причина:**
- Интеграция с Т-Банком
- ЦБ планирует исключить из списка системно значимых
- Неясная судьба программы лояльности

**Статус:** НЕ рекомендуется для включения

## Общие тренды российских банков 2026

### 1. Персонализация
Все топ-банки переходят к модели **выбора категорий пользователем**, отказываясь от фиксированных категорий

### 2. Частота обновлений
- **Ежемесячно:** Сбер, Т-Банк, Альфа, ВТБ
- **Ежеквартально:** ДОМ.РФ (исключение)

### 3. Ограничения
- Лимиты на сумму кешбэка (3,000-30,000₽)
- Количество категорий (3-8 категорий)
- Привязка к подпискам (Pro/Premium tier'ы)

### 4. Геймификация
ВТБ и другие банки внедряют элементы геймификации в программы лояльности

### Архитектура данных

**Таблица: russian_banks**
```sql
CREATE TABLE russian_banks (
  id UUID PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  name_short VARCHAR(50) NOT NULL,
  logo_url TEXT,
  app_link_ios TEXT,
  app_link_android TEXT,
  category_selection_frequency VARCHAR(20), -- monthly, quarterly
  max_categories_free INT,
  max_categories_premium INT,
  base_cashback_percent DECIMAL(5,2),
  max_cashback_percent DECIMAL(5,2),
  is_active BOOLEAN DEFAULT TRUE,
  priority INT, -- 1-5 для сортировки в UI
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```
